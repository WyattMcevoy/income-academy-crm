## How to use

Most of these are markdown worksheets designed to be copy-pasted into:
- A Google Doc / Word doc (for worksheets)
- A Google Sheet / Excel (for trackers — the structure maps to columns)
- Your phone notes app (for checklists you use at sales)

Keep the templates as a template — never edit the originals. Make a copy for each sale / each week / each item.
