# Affiliate Program Application Template

For applying to direct affiliate programs (not networks). Use when a program has an application form requiring a brief pitch, or when reaching out to a company that doesn't advertise a public program.

---

## Template A: Standard application (for programs that accept you based on site quality)

```
Hi [NAME or "Affiliate Team"],

I'd like to apply to the [COMPANY NAME] affiliate program.

About my site:
- Name: [YOUR SITE]
- URL: [URL]
- Niche: [SPECIFIC NICHE]
- Audience: [WHO I SERVE — be specific]
- Monthly traffic: [HONEST NUMBER — ranges OK: "500-1,000/month" is fine]
- Email list: [SUBSCRIBER COUNT]
- Publishing cadence: [HOW OFTEN I POST]

Why I want to promote [COMPANY]:
- I've used [PRODUCT/SERVICE] personally for [TIME PERIOD]
- My audience has asked about [SPECIFIC TYPE OF PRODUCT] frequently
- I see a natural fit with [SPECIFIC ARTICLES/CONTENT I'VE PUBLISHED]

A few articles where I'd naturally include your products:
- [LINK 1] — brief description
- [LINK 2] — brief description  
- [LINK 3] — brief description

I'd love to be considered. Happy to answer any questions.

Best,
[YOUR NAME]
[YOUR EMAIL]
[YOUR SITE URL]
```

## Template B: New site (under 6 months, lower traffic)

Honest about being early, but showing trajectory:

```
Hi [NAME or "Affiliate Team"],

I'd like to apply to the [COMPANY] affiliate program.

My site is newer (launched [X] months ago), so I want to be upfront about traffic and size:
- Site: [YOUR SITE]
- Niche: [SPECIFIC]
- Audience: [WHO]
- Monthly traffic: [HONEST NUMBER — even if small]
- Email list: [HONEST NUMBER]
- Content published so far: [NUMBER] articles

What I bring despite being new:
- Strong commitment to content quality (you can see this in [SPECIFIC ARTICLE])
- Specific niche focus where [COMPANY's product] fits naturally
- Growing audience of [TARGET DEMO]

Why I want to promote [COMPANY]:
- I personally use [PRODUCT] for [SPECIFIC USE]
- I believe it's [SPECIFIC GENUINE REASON]
- I've already mentioned it in [IF YOU HAVE] and will continue to do so regardless of affiliate status

I understand if you only accept established sites. If so, I'd love to reapply in 6 months when I have more traffic and longer track record.

Best,
[YOUR NAME]
[SITE URL]
```

## Template C: Direct company outreach (no published program)

For companies that don't have a public affiliate program but you'd love to work with:

```
Hi [NAME — specific person at the company],

I run [YOUR SITE], a site covering [NICHE] for [AUDIENCE]. I've been using [PRODUCT] for [TIME], and I've mentioned it in [NUMBER] articles.

My audience has been asking about [PRODUCT or CATEGORY] more frequently, and I notice you don't have a public affiliate program. I'd love to propose one.

My stats:
- Monthly traffic: [NUMBER]
- Email list: [NUMBER]
- Audience: [SPECIFIC DEMO]
- Content that naturally features or would feature [COMPANY]: [LINKS]

What I'd propose:
- Standard affiliate commission structure (10-20% for physical products, 20-40% for digital/SaaS is typical)
- 60-90 day cookie window
- Monthly payouts
- Clear terms so we both know expectations

Benefits for you:
- Targeted traffic from [AUDIENCE DEMO]
- I'm happy to write a dedicated review if I haven't already
- Long-term promotion commitment (I don't burn through partners)
- Full FTC compliance on my end

Is this something worth a 15-minute call to discuss? If now's not the right time, any chance of a future program being in the works?

Best,
[YOUR NAME]
[SITE]
[EMAIL]
```

## Template D: Requesting higher commission (for established affiliates)

Once you've been in a program for 3+ months and driving measurable sales:

```
Hi [PROGRAM MANAGER],

I've been an affiliate for [COMPANY] for [X] months. In that time:
- I've driven [NUMBER] referrals and $[REVENUE TO COMPANY]
- My content mentioning [COMPANY] has generated [METRIC]
- [OTHER RELEVANT SPECIFIC METRIC]

I'd like to discuss a commission bump.

Current structure: [CURRENT COMMISSION]
What I'd propose: [SPECIFIC ASK — usually 5-10% higher]

Justification:
- My EPC for [COMPANY] is [SPECIFIC NUMBER if known] vs network average
- I'm a dedicated promoter — 80%+ of my [CATEGORY] recommendations go to you
- I've maintained full FTC compliance and high content quality
- Enhanced commissions would let me prioritize [COMPANY] content over [COMPETITOR]

Would you consider a 3-month commission increase tied to performance? We could reset if I don't maintain it.

Happy to discuss on a call.

Best,
[YOUR NAME]
```

## Tips for success

### Do your homework first
- Confirm they actually have affiliate programs (or search "[COMPANY] affiliate program" to see)
- Read their public terms if available
- Check if they're through a network (ShareASale, Impact, CJ) or direct

### Be honest about your size
- New sites are welcomed by some programs; lying will get you kicked out
- If you're small, focus on audience quality and niche fit

### Show specific relevance
- Generic "I write about [CATEGORY]" is weak
- "I've written 12 articles about [SPECIFIC TOPIC] targeting [AUDIENCE]" is strong

### Link to your best content
- Not everything you've ever written
- 2-4 pieces that show you'd naturally promote them

### Follow up once (not more)
- Wait 10-14 days after first application
- One polite follow-up: "Just checking if this went through — happy to answer any questions"
- If still no response, move on

### Respect rejection gracefully
- Most program managers remember applicants who handled "no" well
- "Thanks for considering. I'll reapply in 6 months when I have more traffic" is better than arguing

---

## What NOT to do

### Don't mass-apply
Personalize every application. Copy-paste is obvious and annoying.

### Don't fake traffic
If you get in via inflated numbers and under-deliver, you'll be dropped.

### Don't complain about commission in the application
Bring commission requests later, once you've proven value.

### Don't burn bridges
Even if you're declined, stay professional. Brands talk to each other.

### Don't forget to send the application thoroughly
Half-baked applications with empty sections get rejected immediately.
