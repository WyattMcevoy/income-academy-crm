# AI Side Income — Prompt Library

Copy-paste-ready prompts for the 5 service lines taught in Module 3. Organized by use case.

## How members use this

1. Browse by folder for your service (writing, bookkeeping, social, customer-service, transcription)
2. Copy the prompt, replace the `[BRACKETED]` placeholders with your client's info, paste into ChatGPT or Claude
3. Always read and edit the output. Never ship AI output raw.
4. Save any adaptations back to your own personal prompt folder
