# Prompts — Writing & Editing

For freelancers offering blog posts, newsletters, product descriptions, and editing services.

## Blog post — from topic to draft

### BW-01: Blog post first draft
```
You are an experienced blog writer for small businesses in the [INDUSTRY] niche.

Client: [CLIENT NAME / TYPE OF BUSINESS]
Audience: [READER — age, situation, what they care about]
Topic: [TOPIC]
Tone: [warm / professional / conversational / authoritative]
Length: [WORD COUNT] words

Write a [LENGTH]-word blog post. Structure:
1. Hook (1-2 sentences that make the reader keep reading)
2. Why this matters right now
3. Main content — 3-5 practical points with short examples
4. What to do next (1-2 sentences)

Use one personal anecdote or specific example to make it human. Avoid cliché phrases like "in today's fast-paced world." Write like you're talking to a neighbor, not an audience.
```

### BW-02: Blog post outline first (iterate before drafting)
```
Before writing, give me a detailed outline for a blog post on [TOPIC] for [AUDIENCE]. Include:
- Working title (3 options)
- Hook (first 2 sentences)
- 3-5 main sections with 1-sentence summaries each
- Call-to-action or conclusion

Wait for me to approve or edit the outline before writing the full draft.
```

### BW-03: Blog post rewrite to match client voice
```
Below are 3 existing blog posts from [CLIENT]:
[PASTE POSTS]

Study the voice, tone, sentence length, vocabulary, use of personal examples.

Now rewrite this draft in that exact voice:
[PASTE DRAFT]

Preserve all factual content. Change only the tone, phrasing, and structure to match the client.
```

### BW-04: Add personal story
```
The draft below is solid but sounds generic. Add ONE personal story from [CLIENT — brief bio] that illustrates the main point. Keep the story under 80 words, grounded in a specific moment, and connected explicitly to the lesson.

Draft:
[PASTE]
```

## Newsletters

### BW-05: Weekly newsletter draft
```
Write a weekly newsletter for [CLIENT NAME], a [BUSINESS TYPE] serving [AUDIENCE].

This week's content:
- Main update: [WHAT HAPPENED / WHAT TO SHARE]
- Upcoming: [EVENT OR REMINDER]
- One personal note: [FOUNDER'S WEEK]

Tone: [warm, like a friend catching you up]
Length: 300-500 words
Format: 3 short sections with subheaders. No salesy language. One soft CTA at the end.
Do not use emojis unless the client's past newsletters use them.
```

### BW-06: Subject line generator
```
Generate 10 subject line options for this email. The audience is [AUDIENCE] and the email covers [TOPIC]. Aim for curiosity without clickbait. Keep each under 50 characters.
```

## Product descriptions

### BW-07: Etsy / shop product description
```
Write a product description for an Etsy shop.

Product: [NAME + BRIEF DESCRIPTION]
Materials/how made: [DETAILS]
Who it's for: [TARGET CUSTOMER]
Tone: [cozy / playful / premium / rustic]
Length: 150-250 words

Structure:
- Opening hook (1 sentence that makes them want it)
- 3-4 key features in short bullet form
- Care instructions or usage tips (1-2 sentences)
- Personal note from the maker (1-2 sentences)

Don't use phrases like "the perfect gift" or "one of a kind." Make each description feel honest and specific.
```

### BW-08: Feature list → benefit-focused copy
```
Here are the product's features (listed by the client):
[PASTE LIST]

Rewrite each one as a benefit statement starting with "You'll..." Focus on what the customer gets, not what the product has.
```

## Editing & polish

### BW-09: Structural edit pass
```
Read the draft below. Do NOT rewrite — just give me 5-8 specific suggestions for improvement. Focus on:
- Structure (does it flow?)
- Clarity (are any sentences confusing?)
- Redundancy (are we repeating ourselves?)
- Energy (where does the writing lose momentum?)

Draft:
[PASTE]
```

### BW-10: Copy edit (no structural changes)
```
Copy-edit the draft below for grammar, punctuation, and clarity. Do not change the structure, argument, or voice. Give me the edited version with changes inline or bolded.

Draft:
[PASTE]
```

### BW-11: Tighten to word count
```
The draft below is [CURRENT WORD COUNT] words. Tighten it to [TARGET WORD COUNT] without losing any substantive content. Remove filler phrases, redundancy, and throat-clearing intros.

Draft:
[PASTE]
```

### BW-12: Grammar + readability check
```
Review the draft below. Flag any:
- Grammatical errors
- Awkward phrasing
- Sentences over 30 words (suggest breaks)
- Passive voice that should be active
- Jargon that could be simpler

Format as a numbered list with the original sentence and your suggestion.

Draft:
[PASTE]
```

## SEO support (for content writers who dabble)

### BW-13: Keyword brainstorm from topic
```
For a blog post about [TOPIC] targeting [AUDIENCE], generate:
- 5 primary keyword candidates (1-3 words)
- 10 long-tail keyword phrases (4-7 words, question-style)
- 5 related topics to link to for internal SEO

Return as a clean table. Don't make up search volumes — just topic ideas.
```

### BW-14: Meta description
```
Write a 150-character meta description for a blog post titled "[TITLE]" covering [TOPIC]. Include the primary keyword naturally. End with a subtle hook.
```

## Client deliverables

### BW-15: Writing brief response
```
The client sent this brief:
[PASTE BRIEF]

Write a response that:
1. Confirms my understanding of the project in 3 bullet points
2. Asks 3 clarifying questions if anything is ambiguous
3. Proposes a timeline (drafts due [WHEN], revisions [WHEN], final [WHEN])
4. Ends with "Does this match your expectations?"

Tone: professional, collaborative, confident.
```
