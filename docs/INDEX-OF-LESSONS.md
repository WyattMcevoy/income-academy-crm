## How to use this file

1. Find the lesson you want to update in the table below
2. Open the matching HTML file (path is shown)
3. Cmd+A to select all, Cmd+C to copy
4. In GHL, navigate to the matching course/module/lesson
5. Click the `< >` "Custom Code Editor" button in the lesson description toolbar
6. Cmd+A to select all existing content, Cmd+V to paste
7. Click Save (modal)
8. Click Save & Publish (lesson)

Estimated time per module: 3-5 minutes.

Total time to update all 36 modules: about 2-3 hours.

You can do this in batches - don't try all 36 in one sitting.
