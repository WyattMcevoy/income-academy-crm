# Income Academy — Course Library

Each course subdirectory is a self-contained product asset library. Everything needed to launch a $47 tripwire + $19/mo continuity + high-ticket back-end stack is here.

## Products in this library

- [`ai-side-income/`](./ai-side-income/) — **AI Side Income Starter Kit** (primary launch)
- [`affiliate-marketing/`](./affiliate-marketing/) — **Honest Affiliate Marketing Starter** (secondary launch)

## Structure (per course)

```
<course-name>/
├── README.md           ← Product overview, pricing, positioning
├── curriculum.md       ← Complete 8-module outline with teaching points
├── prompts/            ← Prompt library (by topic)
├── templates/          ← Ready-to-use templates (contracts, emails, etc.)
├── copy/               ← Landing page copy + Meta ad variants
├── scripts/            ← Sales team scripts, video scripts
└── assistant/          ← Custom GPT / Claude Project config for members
```

## How to use this

**You:** pick a course to launch first. Read the curriculum. Record the talking-head videos following the module outlines. Hire a video editor for post-production.

**Your content team / VA:** use the copy/, templates/, and scripts/ files as starting points. Customize for your voice.

**Your sales team:** use scripts/sales-team.md for phone-sales of the high-ticket back-end.

**Members (via GHL):** get access to the templates/, prompts/, and assistant/ files as part of their $47 purchase. Updated copies drop monthly as part of the $19/mo continuity.

## Compliance reminders

- Never use: "guaranteed," "passive," "push-button," "while you sleep," "automatic," "anyone can"
- Always include: earnings disclaimer, typical-results language, testimonial substantiation
- Have your attorney review before first Meta ad runs. Biz Opp Rule may apply to any offer coupled with an ongoing business relationship (high-ticket coaching, warehouse, etc.)

Content in this repo is the canonical source. When you update a template or add a prompt, commit the change. Members on the $19/mo plan get access to the latest versions via their GHL portal.
