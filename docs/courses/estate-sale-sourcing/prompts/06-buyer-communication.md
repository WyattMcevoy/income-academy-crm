# Prompts — Buyer Communication

Response templates for questions, problems, returns, and reviews.

## ES-50: Answer a "lowball offer" message politely

```
A buyer sent me this message on an item listed at $[LIST PRICE]:
[PASTE THEIR MESSAGE — typically "would you take $X?" where X is
30-50% of list price]

Write me a polite response that:
- Thanks them for reaching out
- Clearly declines their specific offer
- If I'd accept $[MY COUNTER] (assume 10-15% off list), suggests it
  as an alternative
- If I won't take less than list, explains briefly without being
  defensive
- Leaves the door open for them to come back

Tone: respectful, not defensive. Not every lowballer is a jerk —
some are testing.
```

## ES-51: Handle a buyer asking for more photos / info

```
A buyer messaged asking for [SPECIFIC REQUEST — more photos of X,
more measurements, age verification, provenance, etc.].

The item: [ITEM]
What's in the listing already: [SUMMARY OF CURRENT DESCRIPTION + PHOTOS]

Write my response that:
- Provides what they asked for (if reasonable)
- Clearly explains what I can't provide (and why)
- Sets expectations honestly if the answer might not be what they
  hoped (e.g., "I don't have the original box / tags / receipts")

Tone: helpful, not bothered by the question. Buyers who ask smart
questions are often the best buyers.
```

## ES-52: Respond to a return request

```
A buyer purchased [ITEM] for $[PRICE] and is requesting a return
because [REASON THEY GAVE].

My return policy on this listing: [WHAT I STATED — "30-day returns,
buyer pays return shipping" / "as-is, no returns" / etc.]

Evaluate:
- Is their reason legitimate under my stated policy?
- Is it a gray area where I might approve out of goodwill?
- Is it clearly not covered (and if so, how to decline gracefully)?

Then write the response based on your recommendation:
- If approving: clear instructions on how to return
- If declining: empathetic explanation, reference the policy
- If gray area: my counter-offer (partial refund, store credit, etc.)

Tone: professional, not defensive. One good return policy decision
prevents 5 negative reviews.
```

## ES-53: Damaged-in-shipping situation

```
Buyer messaged that [ITEM] arrived damaged in shipping. I packed it
[LEVEL OF CARE — e.g., "with lots of bubble wrap in a double-walled
box"] and insured it for [INSURANCE AMOUNT].

The buyer included [PHOTOS / VIDEO / DESCRIPTION OF DAMAGE].

My goals:
- Keep the buyer happy (preserve my seller rating)
- Get the insurance claim started
- Not lose money if I can help it

Write the response:
1. Acknowledge the problem, empathize
2. Request what I need for the insurance claim (photos of damage,
   photos of box/packing, original shipping label if they have it)
3. Outline next steps (I'll file the claim, they'll get refunded
   as soon as the carrier approves, typical timeline is [DAYS])
4. Offer to escalate if the carrier is slow

Tone: I'm on their team, not adversarial. Damage isn't their fault
and probably isn't mine.
```

## ES-54: Ask for a positive review after good transaction

```
A buyer just received [ITEM] and sent me a nice thank-you message
saying [WHAT THEY SAID — e.g., "it's perfect, packed beautifully,
much appreciated"].

Write me a friendly reply that:
- Thanks them for the message (genuine, not performative)
- Subtly asks for a review if they're happy (eBay star rating /
  Facebook review / whatever platform)
- Invites them to reach out if they're ever looking for similar
  items in the future

Tone: like a conversation with a neighbor. No "please please rate
us 5 stars" nonsense. People who get this message often become
repeat customers.
```

## ES-55: Turn a mildly negative review into a learning moment

```
A buyer left me a [STAR RATING] review on [ITEM]. Their comment:
[PASTE REVIEW]

The underlying issue was [WHAT ACTUALLY HAPPENED — e.g., "listing
said 'minor wear' but they felt it was more than minor"].

Write me:
1. A public reply to the review (other buyers will see it) that:
   - Takes responsibility where appropriate
   - Doesn't blame the buyer
   - Demonstrates I'm reasonable to work with

2. A private message to the buyer offering a resolution (partial
   refund, return, whatever fits)

3. One takeaway for my own improvement — what to change in future
   listings to prevent this.

Tone: professional, no defensiveness, no victimhood.
```
