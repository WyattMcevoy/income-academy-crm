## How to use these

1. Copy a prompt
2. Paste into Claude / ChatGPT / the Income Academy AI Writing Assistant
3. Fill in `[BRACKETS]` with your specific details
4. Iterate — the first answer is rarely the best. Say "make this shorter" or "try a different angle."

Prompt quality improves dramatically with CONTEXT. Don't just say "help me price this." Say "this is a 1960s Pyrex Snowflake Blue casserole, excellent condition except for one tiny chip on the lid rim, I paid $8 — help me price it for eBay based on recent sold listings." Specifics beat clever prompts every time.
