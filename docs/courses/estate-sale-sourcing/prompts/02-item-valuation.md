# Prompts — Item Valuation

30-second value checks to use at sales or when deciding what to buy.

## ES-10: The "is this worth picking up?" check

```
I'm at an estate sale looking at [ITEM DESCRIPTION — be specific:
brand, model, era if known, condition, price].

Quick value check for me:
- Recent eBay sold-listings range (rough estimate based on what you
  know)
- Who actually buys this and why
- Red flags to look for at close inspection (chips, repairs, fakes,
  reproductions)
- Would you buy it at the asked price, and if so, what's my target
  sell-through margin?
- If it's borderline, what negotiation angle would turn a 'meh' into
  a 'yes'?

Fast response — I have 3 minutes to decide before the next buyer
picks it up.
```

## ES-11: Pyrex pattern identifier

```
I'm holding a vintage Pyrex [DISH TYPE — casserole / mixing bowl /
refrigerator dish]. The pattern is [DESCRIPTION — "white with yellow
daisies" or "blue with white snowflakes" or exact pattern name if
I can read it on the bottom].

Identify:
- The pattern name (Snowflake Blue, Butterprint, Gooseberry, etc.)
- Approximate year of manufacture
- Current market value range for a piece in [CONDITION: excellent /
  minor utensil marks / visible chips]
- Any reproductions / fakes to watch out for with this pattern
- Best time of year to list this (some patterns peak at holidays)

If you can't identify from description alone, tell me what additional
details I should look for on the piece.
```

## ES-12: Vintage tools — Stanley / Starrett / Lie-Nielsen

```
I'm looking at a [TOOL TYPE — plane / hammer / measuring tool /
etc.] marked [MAKER + MODEL NUMBER if visible]. Condition:
[CONDITION NOTES — rust, chips, missing parts, etc.].

Evaluate:
- Is this a collector item or a user tool (different buyers, different
  prices)
- Approximate era and market value range for this condition
- Restoration potential — would cleaning/tuning increase value, or
  does the collector market prefer "as found"?
- Common reproductions or fakes to check for
- Best platform to sell (eBay, Reddit r/handtools, specialty dealer)

Red flags I should inspect for close up before I buy?
```

## ES-13: Electronics quick-check (cameras, audio, gaming)

```
I see a [ELECTRONIC ITEM — vintage camera / stereo receiver / gaming
console / etc.] at an estate sale. Asking price: $[PRICE].

Before I decide:
- Is this a known collectible or a generic discard?
- What's the realistic sold range on eBay in [CONDITION] condition?
- What SHOULD I test in the 2 minutes I have (power on, specific
  features, physical condition)
- What's almost certainly broken / missing that I won't discover
  until I get it home?
- Shipping economics — is this size/weight profitable to ship, or
  should I only buy if I can sell locally?

Go / no-go recommendation.
```

## ES-14: The "is this fake" check for collectibles

```
I'm looking at what appears to be [ITEM — e.g., "a Roseville pottery
vase", "a Franz Porcelain bird", "a vintage Coca-Cola sign"]. Asking
price $[PRICE].

In the vintage world, [ITEM CATEGORY] has heavy reproduction
problems. Walk me through the 3-5 specific things I should check to
distinguish an authentic piece from a reproduction/fake, in the
order a trained eye would check them. Include:
- Markings to look for (and red-flag markings)
- Physical details (weight, finish, manufacturing technique)
- Any telltale signs of modern reproduction
- Use of a UV flashlight or other tool if relevant

If I can't verify authenticity in 2 minutes, is this a 'pass' or
still worth the risk at $[PRICE]?
```

## ES-15: The "overlooked gem" prompt

```
I just walked through an entire estate sale and didn't find anything
obviously valuable in my usual categories ([YOUR CATEGORIES]). The
family is asking $[TOTAL PRICE] for everything remaining as a
"clear it all" deal at the end of the day.

Here's what's there:
[LIST WHAT'S LEFT — books, kitchenware, clothing, furniture, random
misc]

Is there ANYTHING in this list that might be quietly valuable and
resellable that I might be overlooking because it's outside my
normal categories? Consider:
- Vintage items with niche collector markets I might not know
- Mid-century modern furniture buried in "just furniture"
- Books (first editions, signed, rare technical)
- Vintage fabric / sewing supplies (hidden collector markets)
- Old paperwork, letters, photos (genealogy / ephemera market)

If nothing stands out, say so honestly — I don't want to buy $200 of
stuff to find $50 of value.
```
