# Prompts — Social Media Ghostwriting

For freelancers writing LinkedIn, Facebook, and other social posts for busy professionals.

## Voice discovery (first client meeting)

### SM-01: Extract client voice from samples
```
I have 5 posts written by my client (or that sound like them). Analyze and summarize their voice:
- Sentence length patterns (short / varied / long)
- Vocabulary level (everyday / professional / academic)
- Tone adjectives (3-5 words: warm? sharp? playful? earnest?)
- Common phrases or verbal tics
- How they open a post (question / bold statement / story?)
- How they close (CTA / reflection / question?)
- Use of specifics (names, numbers, dates) — often or rarely?

Samples:
[PASTE 5 SAMPLES]

Give me a "voice profile" I can refer to when drafting posts for them.
```

### SM-02: Voice-matching draft
```
Using the voice profile below, draft 3 LinkedIn post options on the topic: [TOPIC].

Voice profile:
[PASTE PROFILE FROM SM-01]

Post constraints:
- 150-250 words each
- Must sound like the profile (not like a generic "thought leader")
- One personal detail or example
- End with a soft question or reflection (not a salesy CTA)

Generate 3 distinct options with different angles.
```

## LinkedIn

### SM-03: Week of LinkedIn posts (5 posts from one interview)
```
I recorded a 30-minute interview with my client covering [BROAD TOPIC]. Transcript below.

Extract 5 distinct post ideas. For each:
- Headline/hook (first line)
- Key point
- Supporting example or detail from the transcript
- Closing reflection or question

Then write 5 posts (150-250 words each) using the voice profile:
[PASTE VOICE PROFILE]

Transcript:
[PASTE TRANSCRIPT]
```

### SM-04: LinkedIn post from article
```
Summarize this article into a LinkedIn post from my client's perspective.

Article:
[PASTE OR URL + SUMMARY]

Voice: [PASTE VOICE PROFILE]
Client's angle: [WHY THIS MATTERS TO THEIR AUDIENCE]

Post should be 150-200 words. Start with a hook that makes someone stop scrolling. Don't summarize the article — give the client's take on it.
```

### SM-05: LinkedIn carousel outline
```
Turn this topic into a 7-slide LinkedIn carousel outline.

Topic: [TOPIC]
Audience: [CLIENT'S AUDIENCE]
Goal: [AWARENESS / ENGAGEMENT / LEAD GEN]

For each slide:
- Slide number
- One-line headline
- Body text (under 40 words)
- Visual suggestion (chart, quote, icon, etc.)

End slide 7 with a call-to-action appropriate for [GOAL].
```

## Facebook

### SM-06: Facebook community-style post
```
Write a Facebook post for a local service business client. Goal: community connection, not direct sales.

Client: [NAME + TYPE]
Topic: [WHAT TO POST ABOUT]
Audience: local community, 35-65 demo

Tone: warm, neighborly. Use first-person. Include one specific detail that shows this isn't stock content. End with a question that invites comments (not a sales pitch).

Length: 100-200 words. One emoji maximum.
```

### SM-07: Facebook business update
```
The client wants to announce: [WHAT]

Write a Facebook post that:
- Starts with genuine excitement without being over the top
- Gives the key info in the first 2 sentences
- Adds 1-2 sentences of why this matters to the community
- Ends with a natural next step

Length: 100-150 words. Tone: the voice profile below.

Voice profile:
[PASTE]
```

## Content strategy

### SM-08: Monthly content calendar
```
Build a 30-day content calendar for [CLIENT] posting on [PLATFORM(S)].

Their business: [TYPE + OFFERING]
Posting frequency: [X per week]
Audience: [BRIEF]
Primary goal this month: [AWARENESS / ENGAGEMENT / LEADS / CONVERSIONS]

Structure:
- Mix post types (educational, personal, behind-scenes, customer story, question, promotion)
- Promotion posts no more than 20% of total
- Tie to seasonal or industry events if relevant

Return as a table: Day | Date | Post Type | Topic | Hook/First Line | Voice Note

Don't draft full posts — just the calendar skeleton. Client will approve before I draft each.
```

### SM-09: Repurpose one piece into a week
```
Take this one blog post / video / podcast episode and break it into 5 social posts for [PLATFORM].

Source material:
[PASTE]

For each of the 5 posts:
- Different angle / different key point from the source
- Different post type (quote, story, list, question, lesson)
- Standalone — each should work without the others

Voice: [PASTE VOICE PROFILE]
```

## Engagement

### SM-10: Comment response drafts
```
My client received these comments on their post. Draft responses for each.

Post topic: [TOPIC]
Comments:
1. [COMMENT]
2. [COMMENT]
3. [COMMENT]

Response guidelines:
- Acknowledge the commenter's point
- Add one new insight or small extension
- Don't over-explain or get defensive
- Voice: [PROFILE]
- Length: 1-3 sentences each
```

### SM-11: Handle a negative comment
```
The client received this critical comment:
[PASTE COMMENT]

Post context:
[WHAT THE POST WAS ABOUT]

Draft 2 response options:
- Option A: acknowledge + clarify (no defensiveness)
- Option B: thank + redirect to a private conversation

Neither should argue. Both should close the thread professionally.
Length: 2-3 sentences each.
```

## Short-form (reels, shorts, tiktok scripts for the younger end of demo)

### SM-12: 30-second video script
```
Write a 30-second video script for [PLATFORM] on the topic: [TOPIC].

Audience: [CLIENT'S AUDIENCE]
Voice: [PROFILE]

Structure:
- Hook (first 3 seconds — must make them not scroll)
- Body (15-20 seconds — one idea only, maximum)
- Payoff/CTA (5-7 seconds)

Format: HOOK: [text] / ON-SCREEN ACTION: [describe visual] / BODY: [text] / CTA: [text]

Keep sentences short. Write as if the client will say each line out loud. No jargon.
```

### SM-13: Hook-only brainstorm
```
Generate 15 different opening hooks for a [PLATFORM] post on [TOPIC].

Vary the style:
- Question hook
- Counter-intuitive statement
- Confession / vulnerability
- Number/stat (only if true — don't invent)
- Curiosity gap
- Story opening

Keep each under 15 words. Voice profile: [PASTE]
```

## Recurring / retainer deliverables

### SM-14: Weekly content recap for client
```
At the end of each week, I report back to my client. Write a summary of this week's social activity.

This week:
- Posts published: [LIST WITH TOPICS]
- Total engagement: [LIKES, COMMENTS, SHARES]
- Best-performing post: [WHICH ONE + WHY]
- Notable comments or DMs: [ANY WORTH FLAGGING]
- Plan for next week: [2-3 THEMES]

Format: short email, 200 words max. Tone: confident, concise, collaborative. Close with "Anything you want to adjust?"
```

---

## Voice rules (for the freelancer)

- Your client's voice is the product. Yours disappears.
- Read their last 20 posts before writing the first one for them.
- If in doubt, ask: "Does this sound like something they'd say?"
- Build a voice profile doc for every client in month 1. Update it monthly.
