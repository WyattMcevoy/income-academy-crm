## Converting SVG → PNG/JPG

Most modern platforms accept SVG (browsers, GHL, social media meta tags). If you need raster:

**Free online converters**:
- [cloudconvert.com/svg-to-png](https://cloudconvert.com/svg-to-png) — flexible sizes
- [svgtopng.com](https://svgtopng.com)

**Recommended exports per asset**:
| File | Convert to | Size | Use case |
|---|---|---|---|
| logos/ia-mark-color.svg | PNG | 32×32 | favicon-32.png |
| logos/ia-mark-color.svg | PNG | 180×180 | apple-touch-icon.png |
| logos/ia-mark-color.svg | PNG | 512×512 | GHL square icon |
| logos/ia-horizontal-color.svg | PNG | 600×200 | GHL logo upload |
| course-thumbs/*.svg | JPG | 1280×720 | GHL course thumbnails |
| og-images/*.svg | JPG | 1200×630 | meta og:image tags |
| emails/email-banner-600.svg | PNG | 600×200 | email body banner |
| emails/email-footer-200.svg | PNG | 200×80 | email footer |

**Tip**: Conversion produces a still image of what the SVG renders to. Open the SVG in a browser first to confirm it looks right at the target size before converting.

## How to view these locally

Open any `.svg` file in your browser to see it rendered. Drop into Figma or Illustrator to edit. They're plain XML — you can also open them in any text editor.

## Workflow when assets need updates

1. Edit the SVG (text editor, Figma, or Illustrator)
2. Re-export to PNG/JPG if needed (using free converter above)
3. Re-upload to GHL / wherever
4. Commit the SVG change

Files travel with the repo — version-controlled like code.
